# TOOLS.md — Steward (ops)

## Permitido
- `obsidian` MCP — read-note, create-note, edit-note, move-note, search-vault, add-tags, remove-tags
- `exec` — git commands, backup scripts
- `read` — ler arquivos de sistema e vault
- `cron` — gerenciar cronjobs internos

## Proibido
- `web_search` — ops não faz pesquisa
- `browser` — ops não navega na web
- `mcp-leads` — fora do escopo
- `mcp-crm` (write) — não modifica dados de negócio
- Publicar em canais externos (Telegram/Discord) exceto para erros críticos
