# TOOLS.md — Analyst (researcher)

## Permitido
- `web_search` — Tavily / DuckDuckGo para buscas iterativas
- `web_fetch` — buscar conteúdo de URLs específicas (papers, docs, artigos, GitHub)
- `browser` — sites JS-heavy que bloqueiam fetch direto
- `ctx7` — documentação técnica oficial de libraries e frameworks (usar antes de web_search para tópicos técnicos)
- `read` — ler skills e referências internas (incluindo `skills/deep-research/SKILL.md`)
- `exec` — rodar scripts auxiliares de extração

## Proibido
- `mcp-crm` (write) — não cria/edita contatos
- `mcp-leads` — fora do escopo
- `obsidian` — vault é domínio do ops
- `cron` — researcher não agenda tarefas
- Publicar em canais externos sem aprovação humana
