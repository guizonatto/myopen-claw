# Templates de Resumo — GitHub + Jira

## FAQ Vendas/Suporte
- **O que mudou nesta release?**
- **Há mudanças que impactam o usuário final?**
- **Preciso atualizar algo manualmente?**

## FAQ Usuário Final
- **O que é a nova funcionalidade?**
- **Como ativo essa funcionalidade?**
- **Quais benefícios terei?**
- **Preciso pagar algo a mais?**
- **Descrição da funcionalidade:**

## Resumo Executivo (CEO)
- Data e foco da release
- PRs aprovadas
- Commits
- Linhas adicionadas/removidas
- Resumo dos PRs
- Funcionalidades adicionadas
- Impacto no negócio
- Principais métricas de uso
- Riscos/alertas
- Próximos passos/roadmap
- Reconhecimento de equipe
- Feedbacks recebidos
- Status de SLA/qualidade
- Entregas Jira (issues finalizadas, bugs, features)
