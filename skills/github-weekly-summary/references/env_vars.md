# Variáveis de Ambiente — github-weekly-summary

- GITHUB_TOKEN: token de acesso GitHub (escopo repo)
- GITHUB_REPO: lista de repositórios separados por vírgula (ex: org1/repo1,org2/repo2)
- JIRA_EMAIL: e-mail do usuário Jira (obrigatório para Basic Auth)
- JIRA_TOKEN: API token Jira (obrigatório para Basic Auth)
- JIRA_BASE_URL: URL base da API Jira (ex: https://empresa.atlassian.net)
- JIRA_PROJECT_KEY: chave do projeto Jira (ex: ABC)
- NOTION_TOKEN: token de integração Notion
- NOTION_DATABASE_ID: ID do database Notion
- DISCORD_BOT_TOKEN: token do bot Discord
- DISCORD_CHANNEL_ID: ID do canal Discord para envio
- DISCORD_GUILD_ID: ID Do server do discord que será pego os dados
> Todos obrigatórios para funcionamento completo da skill.
